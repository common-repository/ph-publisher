Load your Performance Horizon publisher credentials for access to PH data. You will be able to quickly add your tracking links to any new post you create.

== Description ==
This plugin is no longer supported. Check out our new plugin for future updates: [Partnerize Partner](https://wordpress.org/plugins/partnerize-partner/)

This will add a settings page to the Admin dashboard so you can provide your Performance Horizon (PH) API credentials. The plugin will then query the PH API to obtain all of your available, and approved, affiliate marketing campaigns. When you add a new post to the site there will be an option in Editor to select your tracking link and add any of the available parameters to the URL.

== Installation ==
Installation of this plugin is easy. Just activate the plugin and visit the Settings page to add your API credentials.
